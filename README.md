:toolbox: Getting Started
Follow these simple steps to run this project locally üëá

Make sure Git and NodeJS are installed on your system.

Clone this repository to your local computer.

```bash
git clone https://github.com/Abhi0019/Space-Portfolio
```
Navigate to the project folder.

```
cd space-portfolio
```
Install dependencies.

```
npm install
```
Run the development server.

```
npm run dev
```
Open your browser and go to:

```
http://localhost:3000
```
Now the app is fully configured and running locally üöÄ
